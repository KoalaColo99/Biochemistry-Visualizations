# Biochemistry Visualizations

Open `index.html` in a browser to use the teaching dashboard.

The tool includes:

- Course-unit navigation for protein structure, enzymes, carbohydrates, lipids, metabolism, and DNA/RNA biochemistry.
- Search and filters for interactive, class-demo, molecular, and pathway visuals.
- Live mini-demos with sliders for amino acid backbone anatomy, chirality, R-group classification, UV absorbance, titration/pI, peptide resonance, dihedral angles, Michaelis-Menten kinetics, chymotrypsin, carbohydrate projections, membrane fluidity, metabolic pathways, and more.
- A saved lesson set stored in the browser, plus print and export options.

Chapter 3 amino acid visuals are original SVG/HTML visualizations built from course learning goals. They are not copied from publisher figures.

To extend the catalog, edit the `resources` list in `app.js`.
