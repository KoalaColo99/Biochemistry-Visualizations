# Biochemistry Visualizations

Open `index.html` in a browser to use the teaching dashboard.

The tool includes:

- Course-unit navigation for protein structure, enzymes, carbohydrates, lipids, metabolism, and DNA/RNA biochemistry.
- Search and filters for interactive, class-demo, molecular, and pathway visuals.
- Live mini-demos with sliders for peptide resonance, dihedral angles, Michaelis-Menten kinetics, chymotrypsin, carbohydrate projections, membrane fluidity, metabolic pathways, and more.
- A saved lesson set stored in the browser, plus print and export options.

To extend the catalog, edit the `resources` list in `app.js`.
